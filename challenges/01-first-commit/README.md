# Challenge 01: First Commit Quest (Easy)

## Objective
Create your first proper commit by fixing a broken intro file.

## Files
- `intro.txt`

## Task
1. Open `intro.txt`.
2. Replace `YOUR_NAME` with your name.
3. Replace `YOUR_BRANCH` with the branch you are currently on.
4. Add one more line: `I am ready to learn Git.`
5. Commit using:
   - `challenge-01: completed first commit quest`

## Hint
Run `git status` before and after committing.

## Success Criteria
- File is updated correctly.
- Exactly one commit is made for this challenge.
