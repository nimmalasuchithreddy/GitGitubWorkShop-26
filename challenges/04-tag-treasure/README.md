# Challenge 04: Tag Treasure (Easy)

## Objective
Mark milestones using annotated tags.

## Files
- `release-notes.md`

## Task
1. Update `release-notes.md` by adding 2 bullet points of features.
2. Commit:
   - `challenge-04: prepared release notes`
3. Create an annotated tag named `v1.0.0-workshop` with message:
   - `First workshop milestone`
4. Show the tag details using `git show v1.0.0-workshop`.

## Success Criteria
- Tag exists and points to the release commit.
- Tag is annotated (not lightweight).
