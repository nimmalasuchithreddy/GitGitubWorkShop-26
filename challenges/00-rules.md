# Challenge Rules

1. Do not delete challenge folders.
2. Use meaningful commit messages.
3. Commit frequently (small commits > one huge commit).
4. If you get stuck, read hints first, then ask your student coordinators.

## Commit Message Format
Use this format for challenge commits:

`challenge-XX: short-description`

Example:
 
`challenge-03: resolved merge conflict in poem.txt`

## Submission
- Push all branches that are part of your solution.
- Ensure your `main` branch is up to date.
- Add your name and roll number in `challenges/student-info.md`.
