# Challenge 02: Branch Rescue (Easy)

## Objective
Practice branch creation and safe isolated work.

## Files
- `bug-report.md`

## Task
1. Create and switch to a new branch named `fix/login-bug`.
2. In `bug-report.md`, replace `STATUS: OPEN` with `STATUS: FIXED`.
3. Add a line: `Fixed by <your-name>`.
4. Commit with message:
   - `challenge-02: fixed login bug on feature branch`
5. Merge this branch into `main`.
6. Delete the branch after merge.

## Hint
Use `git branch` to confirm whether the branch was deleted.

## Success Criteria
- Work is done on a non-main branch.
- Merge is complete.
- Branch is deleted locally.
